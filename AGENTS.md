# Agent Workflow Instructions

## Collaboration Contract (Required)

Work incrementally. Do not execute full end-to-end implementation in one turn.

Required loop:
1. Inspect and summarize findings.
2. Propose a small next step (one slice only).
3. Stop and ask for confirmation: "Proceed with this step?"
4. After implementation, stop again with:
   - what changed
   - verification results
   - next 1-2 options
5. Wait for user approval before continuing.

Rules:
- No multi-file refactors without prior approval.
- No running broad test suites unless approved.
- No docs updates beyond the current slice unless approved.
- If uncertain, ask instead of assuming.
